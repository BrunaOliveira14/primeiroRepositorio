PASSOS PARA HOSPEDAR UM PROJETO NO GITHUN PAGE

  1. Crie um diretório, que será seu repositório local quando você digitar o:
  2. Altere o nome da Branch principal para “main”• git branch –M “main”
  3. Crie um index.html e o style.css
    https://germinatech.github.io/hospedandoSites/
  4. Adicione tudo para o estado preparado
  5. Commite o arquivo no repositório local (criando a versão) • git commit -m “criando os arquivos no repositório local”
  6. Agora vá no GITHUB e crie um repositório remoto
    • Atenção para o nome, vai ser o nome que aparece no seu site público.
    • Lembre de colocar public criar um licence.
  7. No GitHub, vá em Settings > Pages
  8. Em "Branch", selecione main e a pasta / (root). Clique em Save
  9. Após alguns segundos, seu site estará disponível no link exibido em "Pages"
